# Aqua.Tech Phase 1

## Run locally
```bash
npm install
npm run build
```

Then open `index.html` with Live Server.

## Main changes
- Removed Tailwind CDN from production HTML.
- Added local Tailwind build setup.
- Converted `logoo.jpeg` to optimized `assets/logoo.webp`.
- Added `vercel.json` security/cache headers.
- Added `.gitignore`.

## Before deploying
Replace `https://YOUR-DOMAIN.com` in `index.html` after you buy the domain.
